## <center>Installation</center>

### Install hexo-cli
```sh
npm i -g hexo-cli
```
<br />

### Clone Repository and go into folder
```sh
git clone https://github.com/dethMastery/katsuHexoBlog.git
cd katsuHexoBlog
```
<br />

### Install package
```sh
yarn
```
<br />

### Start coding :D
```sh
yarn dev
```
<br />

### Build and deploy
```sh
yarn build
yarn deploy
```
